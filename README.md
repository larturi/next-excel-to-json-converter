# Next 13 Excel to JSON Converter

## Tecnologies

- Next 13.5.4
- Node 18.14.1
- Next Auth
- Google Auth
- Prisma
- MongoDB

## Getting Started

First, run the development server:

```bash
nvm use 18.17.0

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Prisma

```bash
npx prisma db push
npx prisma generate
```
